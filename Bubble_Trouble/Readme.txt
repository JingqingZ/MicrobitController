This folder describes how to implement generic microbit controllers and a server for the online game Bubble Trouble.

1-Instructions: Provides a detailed guide on how to set-up and use the controllers on a Windows Machine.

2- Controller_Microbit(s).py: A python script to be downloaded onto the Micro:bit(s) to be used as controllers.

3- Server_Microbit.py: A python script to downloaded onto the Micro:bit to be used as the receiver/server.

4- Game_Control_parser.py: A python script to be run on the PC that maps the controls captured in the receiver to the needed
keyboard commands.
